# Swoole Unit

Swoole Unit allows you to run PHPUnit tests on Swoole.

## Usage

You may install Swoole Unit via Composer:

```sh
$ composer require --dev huang-yi/swooleunit
```

Then, you can run your tests:

```sh
$ ./vendor/bin/swooleunit
```
